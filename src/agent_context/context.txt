    *   **特征峰举例 (FTIR)**: 其红外光谱特征与有机缓蚀剂相似。例如，迷迭香提取物的主要活性成分在以下位置有显著吸收峰：**3370 cm⁻¹** (-OH)，**1695 cm⁻¹** (C=O)，**1606 cm⁻¹** (芳香环 C=C)，以及 **1280 cm⁻¹** (C-O)。这些官能团参与了在金属表面的吸附过程。

### 4. 复合缓蚀剂

复合缓蚀剂由两种或以上缓蚀剂组分复配而成，利用各组分间的协同效应，实现优于单一组分的缓蚀效果。分析其光谱活性时，需分别考虑各组分的贡献。

*   **实例1：有机胺 + 钼酸钠 (有机/无机复合)**
    *   **作用机理**: 结合了有机胺的吸附成膜和钼酸盐的钝化成膜双重机理。
    *   **光谱活性分析**:
        *   **有机胺组分**: 具有**红外活性**，可在 **3300-3500 cm⁻¹** 区域观测到-NH特征吸收峰。
        *   **钼酸盐组分**: 具有**拉曼活性**，可在 **896 cm⁻¹** 附近观测到Mo-O对称伸缩振动的强散射峰。
    *   **总结**: 该复合缓蚀剂整体上是**红外和拉曼双重活性**的，可以通过不同的光谱技术分别观测到两种组分的特征，判断它们是否都参与了成膜。

*   **实例2：六偏磷酸钠(SHMP) + 氨基三亚甲基膦酸(ATMP) (无机/有机复合)**
    *   **作用机理**: 利用无机磷酸盐和有机膦酸的协同作用形成更稳定的保护膜。
    *   **光谱活性分析**:
        *   **SHMP组分 (无机)**: 具有**红外活性**，可在 **1282 cm⁻¹ (P=O)** 和 **877 cm⁻¹ (P-O-P)** 处观测到其特征峰。
        *   **ATMP组分 (有机)**: 具有**红外活性**，其亚甲基(-CH₂)和膦酸基团(-PO₃H₂)在红外光谱中也有相应的特征吸收峰。
    *   **总结**: 该复合体系整体具有**红外活性**。通过分析形成的膜层，可以同时观测到来自SHMP和ATMP的特征峰，证明两种组分均参与了成膜过程。
