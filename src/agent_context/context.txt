### **核心输入信号**

*   **信道状态信息 (Channel State Information, CSI):** 这是目前非接触式Wi-Fi感知领域最主要的输入信号。CSI能够详细描述无线信号在传播路径中经历的衰减、多径等信道特性。当环境中有人体活动或姿态变化时，会引起CSI数据的波动，深度学习模型通过分析这些波动模式来实现感知。

### **最先进的算法策略与框架**

目前业界已经提出了一系列先进的深度学习算法和框架，旨在提升感知的准确性、鲁棒性和效率：

*   **Attention Based BLSTM (基于注意力的双向长短期记忆网络):** 这是一种先进的神经网络模型，专门用于被动式的人体活动识别。它能够有效捕捉CSI时间序列中的长期依赖关系，并通过注意力机制聚焦于最关键的信号特征。
*   **DeepSeg:** 这是一个基于深度学习的活动分割框架。与简单地识别整个时间段内的单一活动不同，DeepSeg可以对连续的CSI信号流进行分割，并识别出其中包含的多个不同活动。
*   **WiADG (Adversarial Domain Adaptation):** 该策略采用无监督的对抗性领域自适应技术，旨在解决设备异构性带来的挑战。通过这种技术，模型可以实现稳健、设备无关（device-free）的手势识别，增强了算法的通用性和鲁棒性。
*   **EfficientFi:** 此方法着眼于解决大规模部署和轻量化应用的需求。它通过对CSI信号进行压缩，在保证感知性能的同时，降低了数据存储和计算的开销，使得大规模、轻量级的Wi-Fi感知成为可能。
*   **One-shot Learning:** 这是一种学习策略，旨在通过极少量（甚至一个）的样本来识别新的活动类别，从而提高人体活动识别系统的适应性和快速部署能力。

**1. SenseFi：综合性深度学习框架**
SenseFi是一个基于PyTorch构建的、开源的综合性Wi-Fi感知深度学习框架和库 [1, 2, 8]。它旨在标准化和推动基于深度学习的Wi-Fi感知研究，支持多种感知任务，如人类活动识别、手势识别和跌倒检测等 [1, 8]。

为了训练和评估感知算法，研究人员开发了像EHUNAM这样的综合性CSI数据集 [3, 4, 8]。该数据集使用Intel 5300网卡在实验室环境中采集，包含了10名参与者执行行走、跑步、坐下、站起、跌倒以及空房间等六种活动的原始CSI数据，为算法性能评估提供了基准 [4, 5, 6]。

**输入信号**：
当前最先进的算法策略普遍使用从Wi-Fi信号中提取的**信道状态信息（CSI）**作为输入 [1, 8]。

**算法策略**：
